CSS文档流

在文档流中，内联元素从左边开始一个接一个地显示，块元素从顶部开始向下显示并移动页面
div是块级元素他会独占一行有多个div他会从上往下排，就是上下流动
span是内联元素他会从左向右排，有多个span时也不会换行，直到内联元素长度大于父元素的长度才会换
块级元素(block)：独占一行，高度(height)、宽度(width)、内补白(padding)、外部白(margin)都可以控制
内联元素(inline)：会和相邻的内联元素同处一行， 高度(height)、宽度(width)、内补白(padding-top/bottom)、外部白(margin-top/bottom) 都不可变margin和padding的left和right是可变的，高度和宽度是由其文档流元素的高度和宽度决定的。
内联块级元素(inline-block)：可以设置宽高，但是不会占一行。

CSS盒模型

CSS有两种盒模型： 1 content-box 宽度为内容区（content）的宽度，不包含 padding 和 border；
2 border-box 宽度为 内容区（content） + padding + border 的总和。

CSS布局

布局分类 固定宽度布局：一般宽度为960/1000/1024px，你再怎么宽我也不宽，你比我小了我就小一点，适用于电脑布局。
不固定宽度布局：主要靠文档流的原理来布局，屏幕在宽我的图标也就变大一点，平铺屏幕，适应于手机布局。

1 float布局

再给元素添加float时，元素会脱离文档流，而div的高度是由内部文档流的高度决定的， 都float了就没有高度了，所以要再float元素的父元素上加clearfix 

      clearfix::after {
          content: '';
          display: block;
          clear  : both;
      }
负margin 在做平均布局时右外补白会多出一块很蛋疼把元素用一个div套起来给一个负margin就可以解决， 就相当于向外扩展出了一段预留空间，来容纳多余的补白。 两栏布局：顶部条 三栏布局：内容区 四栏布局：导航 平局布局：产品展示区

2 flex布局

两栏布局：顶部条 三栏布局：内容区 四栏布局：导航 平局布局：产品展示区

3 grid布局

两栏布局：顶部条 三栏布局：内容区 四栏布局：导航 平局布局：产品展示区

CSS的渲染原理

浏览器得到一个html页面浏览器开始构建HTML树(DOM TREE),遇到CSS样式会构建CS树(CSS RULE TREE) , 然后两束合并成渲染树（render tree） 最后，渲染树构建完成后会经过三个步骤： Layout（布局）文档流、盒模型、计算大小和位置，Paint（绘制）把边框的颜色、文字的颜色、阴影等画出来， Compose（合成）根据层叠关系展示画面。

transfrom常用功能

常配合transtion过度，inline元素不支持transform需要先变成block。 位移：translate 缩放：svale 旋转：rotate 倾斜：skew 一般用translate(-50%，-50%)做绝对定位的居中元素 两次transfrom实现动画 transtion：属性名 时长 过渡方式 延迟 并不是所有元素都可以过度display：none到block 没法过度 一般写成visibility:hidden => visible
