HTML常用标签

a标签

属性：

  1. href:这里写地址

  2. target:这里写怎么跳转

    "_blank"(新开一个页面跳转)
    "_top"(在顶级页面跳转)
    "_self"(在自己这跳转)
    "_parent"(在自己上一级跳转)

  3. download:加上这个就能下载了

  4. target"xxx":在xxx这个窗口打开，没有就新创建一个xxx窗口再打开

作用：

  1. 可以跳转外部页面
    href="baidu.com"(点我去百度)
  2. 跳转内部锚点
    href="#xxx"(去本页id=xxx的地方)
  3. 跳转邮箱
    href="mailto:123@qq.com"(发邮件给123)
  4. 跳转电话
    href="tel:12333334444"(打电话给123)

特殊的需求:我要一个点了之后什么也做的东西

  • href="" (刷新当前页面)
  • href="#" (不刷新返回页面顶部)
  • href="javascript:;" (真什么也不会做，特殊的需求解决了)

table标签

相关的标签

  • table:表格标签
  • thead:表头标签
  • tbody:表的身体
  • tfoot:表的底部
  • tr:行，表行
  • td:表体
  • th:表头，会加粗
table的相邻有空隙
加上边框border:1px solid red;
让中间没有空隙border-collapse:collapse;(合并，默认不合并)
border和border之间的距离border-spacing:0;

img标签

作用
会发送一个get请求，展示一张图片
属性
alt:加载失败时显示的文字
width:写宽度高度自适应
heigth:写高度宽度自适应
src:图片源
注:宽度高度都写会变形(不建议都写)永远都不要让图片变形！

from标签

作用
会发送一个请求，然后刷新页面
属性
action: action="/123" 后面加请求的地址写baidu.com或xxx.com就会发请求去百度或xxx，
写123或xxx页面就会变成123或xxx
autocomplete:在form上加一个autocomplete="on"会在输入框给我们一些建议
method:"get" "post" 请求方式
target: target="a"(我要提交给a)

input标签

作用
让用户输入内容，在form标签里会把输入的内容提交

属性

  • type="chackbox"    复选框 给他们同一个name
  • type="button"       一个按钮
  • type="text"        文本框
  • type="password"    密码框 都是小圆点
  • type="color"     可以选颜色
  • type="submit"     提交按钮 不写值也会自动显示一个提交
  • type="file"      可以选择文件上传，但是只能传一个，加一个multiple就能多个选多个文件了
  • type="radio"     单选框，name值一样的只能选一个
  • type="hidden"     隐形的，你看不见我，但是我的数据该提交还是提交
  • textarea      右下角可以拖拽大小不固定，"resize:none; width:50%; hightyoxia:50%;"三连一下就好了，固定了
  • select     下拉列表 option value="1" option标签写在select里面写几个下拉列表有几个
  • required:     有了我你必须写点东西，不写不让提交哦
